## Tips for Consistent Results

### Always Include

1. Art style definition (watercolor, hand-drawn)
2. Background color specification (#F3F1EA for cream)
3. Brand color references (#FF2D7A, #1D7C5A)
4. "No text" to avoid unwanted labels
5. Composition guidance (centered, wide, etc.)

### Avoid

1. Vague style descriptions
2. Multiple subjects that could confuse
3. Asking for text or labels
4. Complex scenes with too many elements
5. Conflicting style directions

### Iterating on Results

If the first result does not match your vision:

1. Identify what was wrong (color, style, composition)
2. Add more specific keywords addressing that issue
3. Try different phrasings of the same concept
4. Use negative prompts if available ("not realistic, not 3D")

### Maintaining Consistency

When generating multiple related assets:

1. Copy your base prompt exactly
2. Only change the subject description
3. Keep all style and color keywords the same
4. Generate in batches for better matching
5. Re-run failed generations rather than accepting "close enough"

## Post-Processing Recommendations

After generating images:

1. **Background Removal:** Use remove.bg or similar for icons
2. **Color Correction:** Adjust to match exact brand colors
3. **Resizing:** Scale to needed dimensions
4. **Compression:** Reduce file size for web
5. **Format:** Save as PNG for transparency, WebP for size
